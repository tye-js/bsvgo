3. **收藏功能测试**

## 📝 使用说明

### 开发环境启动
```bash
npm run dev
```

### 数据库操作
```bash
npm run db:push    # 推送 schema 到数据库
npm run db:studio  # 打开数据库管理界面
```

### 构建生产版本
```bash
npm run build
npm run start
```
